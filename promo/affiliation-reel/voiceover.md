# Voice-over Reel Affiliation — Script ElevenLabs

## Réglages recommandés

- **Voix** : Antoine ou voix FR masculine grave et chaude (test plusieurs, prends celle qui sonne pro sans être radio commerciale)
- **Stabilité** : 50
- **Clarity + similarity** : 75
- **Style exaggeration** : 30 (un peu d'expressivité, pas plat)
- **Speaker boost** : ON
- **Modèle** : Multilingual v2 (gère bien le français)

À copier-coller dans ElevenLabs **avec les espaces et la ponctuation exacte** (ils servent au phrasé).

## Musique de fond (optionnel)

Une nappe sobre style "Lo-fi piano" ou "Cinematic ambient" à -20 dB. Évite la musique trop rythmée qui distrait du message.

Sources libres de droit :
- **Epidemic Sound** (abonnement, qualité top)
- **YouTube Audio Library** (gratuit, "Cinematic" → "Inspirational")
- **Pixabay Music** (gratuit, chercher "ambient piano")

## Mixage suggéré

- Voice-over : 0 dB (volume normal)
- Musique : -22 dB (très en retrait)
- Ducking : -6 dB sur la musique quand la voix parle (optionnel)
